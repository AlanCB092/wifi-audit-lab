# 🛡️ WiFi Audit Lab

Plataforma profesional de auditoría WiFi con dashboard en tiempo real.

## ✨ Características

- 🔍 **Escaneo de redes WiFi** - Detecta SSID, BSSID, canal, señal y cifrado
- 📊 **Dashboard interactivo** - Visualización de métricas de seguridad
- 🗺️ **Mapa de amenazas** - Geolocalización de ataques globales
- 📱 **Detección de dispositivos** - Identifica equipos en tu red
- 🚨 **Alertas en tiempo real** - Notificaciones de eventos de seguridad
- 📈 **Análisis de tráfico** - Monitoreo de paquetes y ancho de banda

## 🛠️ Tecnologías

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Python, FastAPI
- **Base de datos**: Redis
- **Visualización**: Recharts, Folium
